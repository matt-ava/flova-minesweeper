# Flova Minesweeper Minigame

The project is under development, already playable but not yet integrated with web3.

## Requirements
<pre>
NodeJs version > 14
</pre>

## Installation
<pre>
npm i
</pre>

## Play
<pre>
node index.js
</pre>
Open the browser and navigate to 127.0.0.1:10000
